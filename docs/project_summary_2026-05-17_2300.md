# 眼斑双锯鱼鱼体检测与多目标跟踪项目阶段记录

> 整理时间：2026-05-17 23:00  
> 当前状态：已完成通用模型测试、fish/reflection 双类别检测基线、V2 训练尝试，以及加入 TOP_test_2min 样本后的 V3 阶段验证。  
> 说明：本文档记录的是 2026-05-17 晚间可总结出的项目阶段成果，不包含 2026-05-18 之后的 V4/V5 训练结论。

## 项目目标

本项目面向眼斑双锯鱼群体行为分析，目标是从顶视角视频中检测真实鱼体、区分鱼缸壁反射，并进一步使用多目标跟踪方法生成鱼体二维轨迹，为后续速度、距离、群体结构和行为分析提供数据基础。

整体流程如下：

视频输入  
→ 视频预处理  
→ fish / reflection 双类别检测  
→ 多目标跟踪  
→ 轨迹导出  
→ 行为指标分析

## 当前任务

当前视觉任务主要包括：

- 真实鱼体检测
- 鱼缸壁反射识别
- 多目标跟踪 ID 分配
- 轨迹稳定性初步评估

当前检测类别：

| 类别 ID | 类别名 | 含义 |
|---:|---|---|
| 0 | fish | 真实鱼体 |
| 1 | reflection | 鱼缸壁上的完整鱼形反射 |

## 当前进展

截至 2026-05-17 23:00，项目已完成以下阶段：

1. 使用通用 YOLO 模型进行初步测试，确认通用 COCO 模型不能直接用于本任务。
2. 构建 fish/reflection 双类别检测数据集。
3. 训练早期 fish/reflection 检测基线模型。
4. 尝试 V2 训练，发现单纯增加训练轮数并不能根本解决困难样本问题。
5. 加入 TOP_test_2min 中约 200 张人工标注样本，进入 V3 阶段。
6. 使用 ByteTrack 进行初步多目标跟踪测试。
7. 初步判断后续应继续扩充困难样本，尤其是反射、气泡遮挡、鱼体重叠和边缘漏检场景。

## 当前主要结论

当前阶段最重要的判断是：

- 通用 COCO 预训练模型不适合直接检测眼斑双锯鱼。
- fish/reflection 双类别建模是必要的。
- 数据质量比单纯增加训练轮数更关键。
- TOP_test_2min 的新增样本对检测完整性和跟踪稳定性有明显帮助。
- 后续需要继续通过人工标注和模型预标注扩充困难样本。
- 多目标跟踪的核心问题不是单独的检测精度，而是检测稳定性、漏检、遮挡和 ID 互换的综合结果。

## 后续计划
